# Muscadine Finance - Marketing Site

The marketing and information website for Muscadine Finance, featuring a Roman aesthetic design.

## Features

- Roman-inspired design with gold and stone color scheme
- About section explaining Muscadine Finance's mission
- What We Do section describing DeFi vault operations
- Contact information and Bitcoin services
- Call-to-action linking to the DeFi dashboard

## Tech Stack

- **Framework**: Vite + React + TypeScript
- **Styling**: Tailwind CSS
- **Routing**: React Router DOM
- **Fonts**: Playfair Display (Google Fonts)

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This project is configured for Vercel deployment. The site should be deployed to:
- `muscadine.box`
- `www.muscadine.box`
